# banner-for-wordpress
Banner plugin for your WordPress website.

# About a plugin
A plugin that will allow you to display a vertical banner on your website with interesting features.

# How to install?
To install the plugin, download the zip archive from this repository, upload it to your web hosting or virtual machine, and then extract the archive into /wp-content/plugins/ (it should be like this: /wp-content/plugins/vertical-banner/vertical-banner.php).

Go to the site admin panel and in the plugins section, find the installed plugin and activate it.

You can customize and change the banner however you like.
